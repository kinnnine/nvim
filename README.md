## Quickstart

make sure to backup old configs.

cmd<br/>
`git clone https://github.com/kinnnine/nvim %localappdata%\nvim`

powershell<br/>
`git clone https://github.com/kinnnine/nvim $env:LOCALAPPDATA\nvim`

post-install<br/>
execute `:PlugUpdate` inside nvim, wait for the plugins update and then restart.

## Requirements

- [Neovim](https://neovim.io/doc/install/) 0.11.6+
- [Neovide](https://neovide.dev/installation.html) 0.15.2+
- ripgrep
- win32yank
- [nerd-font (CommitMono)](https://github.com/ryanoasis/nerd-fonts/releases/download/v3.4.0/CommitMono.zip) 3.4.0+

### winget: ripgrep

`winget install ripgrep`

### winget: win32yank

`winget install win32yank`
